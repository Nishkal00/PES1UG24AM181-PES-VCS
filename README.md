# PES1UG24AM181-PES-VCS

## Operating Systems Assignment

Implementation of a simplified version control system in C using phase-wise development and Git commit tracking.

---

## Objective

The goal of this assignment is to understand core version control concepts by implementing a lightweight VCS with support for:

- Object storage
- Tree structures
- Index / staging area
- Commit creation
- History tracking
- Incremental development using Git

---

## Technologies Used

- C Programming Language
- GCC Compiler
- Makefile
- Git & GitHub
- Linux Mint

---

## Project Structure

- `object.c` - Handles object read/write and hashing
- `tree.c` - Manages tree parsing and serialization
- `index.c` - Handles staging/index operations
- `commit.c` - Creates commits and updates history
- `Makefile` - Build automation

---

make
